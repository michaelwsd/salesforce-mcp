Cost summary — what this engine actually adds

  • Already paid for: Salesforce, Microsoft 365, GitHub, the scraper — the bulk of the stack carries no marginal cost.
  • Net new spend: Render hosting (~US$7–25/mo) + Claude API usage (cents per prospect) + any paid scraping infra — order of tens of dollars/month.
  • Potential saving: retiring HubSpot (under review) likely exceeds the entire new running cost of this engine.
  • The real external cost is Tecala — the mandatory data gate, separate from the engine's running cost. Confirm its commercial terms first.
